# Instalação Quiosque IEAFE BE

## Requisitos

- Node.js instalado
- Banco de dados configurado
- Navegador atualizado

## Instalação

1. Baixar o projeto

2. Instalar dependências:

npm install

3. Iniciar servidor:

npm start

4. Abrir:

frontend/index.html


Sistema:
Quiosque IEAFE BE
Igreja Evangélica Avivamento da Fé Bélgica
